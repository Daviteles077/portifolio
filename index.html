Jupyter Notebook (.ipynb)

Atividade 2 — Exercícios de Limpeza, Transformação e Agrupamento de Dados

Arquivo: pandas_intro_ex2.ipynb

Manipulação avançada e estruturação analítica de emendas parlamentares brasileiras via biblioteca Pandas, utilizando o conjunto de dados emendas_parlamentares.parquet contendo 29 colunas estruturais (ex: Código, Ano, Tipo, Valor Empenhado, Liquidado, Pago e Regiões).

Consultas Complexas Implementadas:

Query 1: Filtragem de emendas orçamentárias relativas ao ano de 2022. Retornou um total exato de 6.108 emendas registradas no período.

Query 2: Filtro condicional da Região Norte associado à restrição de Valor Empenhado > 500.000 para mapeamento de investimentos de alto escalão.

Pandas Parquet
import pandas as pd

# Carga de dados em formato colunar Parquet
df = pd.read_parquet('emendas_parlamentares.parquet')

# 1) Filtragem por ano específico
df_2022 = df[df['Ano da Emenda'] == 2022]
print(f"Total 2022: {df_2022.shape[0]}") # Output: 6108

# 2) Filtro condicional multi-variável de Região e Valor
df_norte_alta = df[(df['Região'] == 'Norte') & (df['Valor Empenhado'] > 500000)]

Atividade MER Estacionamento — Modelo Relacional

Arquivo: Atividade MER estacionamento.xml

Mapeamento estrutural detalhado e modelagem lógica de banco de dados relacional para controle operacional de um estacionamento utilizando o software **DB Designer 4** com tipagem nativa para o SGBD **MySQL**.

Detalhes Técnicos da Arquitetura:

• Entidades Mapeadas: Estruturação de tabelas cruciais como tb_funcionario (gerenciamento de funcionários), contendo restrições de integridade, chaves primárias automáticas e mapeamento de chaves estrangeiras.

• Regras de Relacionamento: Definição de gatilhos automáticos relacionais estruturados (ex: Rel_04 e Rel_07) contendo cláusulas rígidas de OnDelete=3 (Cascade/No Action) e OnUpdate=3 para garantir consistência sistêmica total.

Média Dinâmica de N Números

Arquivo: ex_01_3_whi_tu_cdml_22610699 _Davi_Teles.py

Algoritmo interativo que solicita ao usuário a quantidade dinâmica de valores a serem computados, executa um loop de acumulação com validação sequencial e apresenta a média aritmética final formatada com duas casas decimais.

While Loop
N = int(input('Digite a quantidade de números '))
soma = 0
contador = 0

while contador < N:
numero = float(input(f'Digite o {contador+1} número: '))
soma += numero
contador += 1

media = soma / N
print(f'A media dos {N} números é: {media:.2f}')

Conversor de Temperatura Celsius / Fahrenheit Estático

Conversão Algorítmica

Conversor Termométrico Customizável por Escopo

Evolução do conversor termo-dinâmico que permite ao usuário estabelecer as balizas de início e fim da amostragem em Fahrenheit, controlando o processamento matemático do laço de repetição condicional de forma dinâmica.

Escopo Dinâmico
print("CONVERSAO FAHREINHEIT – CELSIUS")
print("CELSIUS FAHREINHEIT")
inicio = int(input("Digite o valor inicial em Fahrenheit: "))
fim = int(input("Digite o valor final em Fahrenheit: "))
f = inicio

while f <= fim:
c = (5/9) * (f - 32)
print(f"{c:8.2f} {f:8.2f}")
f += 1

Cálculo de Médias com Laço de Repetição Definido (For)

Arquivo: ex_04_3_whi_tu_cdml_22610699 _Davi_Teles.py

Algoritmo clássico de iteração que gerencia a entrada de notas acadêmicas sequenciais para uma turma delimitada fixa de 4 alunos, utilizando um laço estruturado for i in range() com indexação visual limpa.

For Loop Iteration
soma = 0
N = 4

for i in range(1, N+1):
nota = float(input(f"Digite a nota do aluno número {i}: "))
soma += nota

media = soma / N
print(f"Média da turma = {media:.2f}")
